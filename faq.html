FAQ

Common questions about how Calc-HQ works.

What is Calc-HQ?

Calc-HQ is the hub for a group of financial and operational calculators. It helps visitors navigate to the right live tool by category.

Does Calc-HQ store my information?

No. The hub is a static site. Linked calculators are designed to run locally in the browser and should be reviewed individually for their own methodology and policy pages.

Why use a category filter?

The filter keeps the hub readable as more live tools are added. It reduces clutter while still relying on one network file as the source of truth.

Are these tools financial advice?

No. They are informational calculators and estimates. Important decisions should still be verified with official sources or qualified professionals.
